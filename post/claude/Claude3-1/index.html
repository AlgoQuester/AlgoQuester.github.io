【实测】Claude3注册教程：解决Claude3 Opus被封无法发送手机验证码问题

发表于2024-03-06|更新于2024-04-08
|阅读量:

Anthropic 3月4日宣布推出最新的大型语言模型（LLM）系列——Claude 3，该系列模型在各种认知任务上树立了全新的性能标杆。Claude 3 系列包括三个子模型：Claude 3 Haiku、Claude 3 Sonnet 和 Claude 3 Opus，每个模型都提供了不同程度的智能、速度和成本选择，旨在满足广泛的人工智能应用需求。

许多用户在尝试注册Claude3时遇到了一个常见障碍——无法接收手机验证码。这一问题尤其困扰着那些没有海外手机号的用户。

不过幸运的是，有一种解决方案可以帮助你成功完成注册过程。本文将详细介绍如何利用Wildcard服务提供的海外手机号进行Claude3注册。

1. Claude 可在哪些地区使用？

Opus 和 Sonnet 目前已整合到 Anthropic 的 Claude.ai 和 Claude API 中，覆盖了 159 个国家和地区。Haiku 很快也将推出。

此外，Sonnet 还可以通过亚马逊云科技的 Bedrock 服务以及谷歌云 Vertex AI Model Garden 的私人预览渠道获取，而 Opus 和 Haiku 也将很快登陆这两个平台。

2. Wildcard是什么？

Wildcard 是一个专门为 AIGC（例如 chatgpt、midjourney、claude 等）提供虚拟信用卡和手机号服务的平台，支持多个国家和地区的手机号。

这些虚拟手机号可用于接收短信验证码，非常适合需要验证国外手机号以注册 Claude 3 Opus的情况。

3. 注册 Claude3 教程

Step1：获取 Wildcard 海外手机号

  1. 访问 Wildcard （通过链接进入可享受2美元优惠）。
  2. 根据网站指引注册账户。
  3. 进入网站，找到海外手机号菜单栏，点击“立即申请”。

  1. 每个账户有三次免费申请机会，但即使使用了手机号，也可能会失败。

Step2：使用获得的手机号注册Claude3

  1. 打开 Claude3 注册页面，填写邮箱等信息。
  2. 输入从 Wildcard 获得的海外手机号。
  3. 点击发送验证码。

Step3：接收并输入验证码

  1. 返回 Wildcard 界面。
  2. 检查你的虚拟手机号收件箱，找到来自 Claude3 的短信验证码。
  3. 将验证码输入到 Claude3 的注册页面中，完成验证。

4. 结语

Anthropic的Claude 3系列的发布代表着在大型语言模型领域取得了重要突破。

这一系列在智能水平、处理速度和多样化应用方面表现出色，预示着人工智能技术的新发展方向。

随着Claude3系列模型的进一步应用和优化，我们有理由相信，人工智能的未来将更加多元和智能。

使用Wildcard所提供的海外手机号可以解决无法接收手机验证码的问题，从而完成Claude3的注册过程并享受其提供的服务。
